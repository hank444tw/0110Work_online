# SOSO部落格
> _自主練習_   

* 功能說明
  1. 閱讀部落格文章。
  2. 頁面互動效果(Ex:大事記頁面的時間軸效果)。
  3. 線上撰寫文章，上傳文章圖片。
  4. 文章管理CRUD。
  5. RWD響應式網頁。
 
* 使用技術
  1. ASP.NET MVC
  2. C#
  3. js
  4. css
  5. Ajax
  6. Entity framework
  7. Linq

* 架站技術
  1. iis
  2. Dns
  3. ssl
  4. Google Search Console Tools
  
* 網站架構
  |        | 說明 |程式 |
  |------- |:-----|:------:|
  | **前端**   |  1. 具有兩條`navbar`，透過撰寫`js`程式，當頁面寬度小於`992px`，則轉換為手機平板用的。 <br>2. 當頁面寬度小於`768px`，轉換為直式的`banner`。</br> 3. 透過`css`和`js`實現一些頁面動態的效果(Ex:`css`的`transform`屬性、`js`的`事件監聽`)</br>4. 使用`aos`套件搭配`js事件監聽`實現大事記頁面區塊飛升效果。</br>5. 使用`ckeditor`文章編輯器套件，再`js`驗證圖片上傳格式，和取得文章內容回傳後端。 |  [程式碼](https://github.com/hank444tw/0110Work_online/tree/master/0110Work/Views) |
  | **後端**   |  1. 第  |  [程式碼](https://github.com/hank444tw/0617Work/blob/master/0617Work/Controllers/HomeController.cs) |
